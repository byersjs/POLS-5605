What are the Course Goals?

  • This is a very applied class
    • We are going to learn practical skills
  • From the very beginning, we will use

    • The programming language R

    • To analyze real data

    • For the purpose of answering substantive political questions

  • In the process, you will learn

    • Statistics

    • How to code in R (R is the name of the program and the name of the language)

  • In this course you will learn how to
    • Analyze data and
    • Evaluate someone else’s data analysis
  • Why do political scientists need to learn how to analyze data and evaluate data analyses? (Why are you taking POLS 5605?)
    • Most political decisions are made based on data
    • If you cannot do the analysis yourself, you need to at least know how to distinguish a good quantitative study from a poorly conducted one

Statistics and Coding

  • For the purpose of analyzing data, you are going to have to learn some statistics and coding
  • Do not panic! The course is setup so anyone can do well. I assume no prior knowledge of either
  • Good news! These are very practical skills, currently in high demand for well-paid jobs
